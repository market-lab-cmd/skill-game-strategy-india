# Project Specification

## Positioning
Educational documentation repository.

## Allowed
- Gameplay explanations
- Strategy notes
- Beginner guides

## Forbidden
- Gambling / betting content
- "earn money" language
- Promotional or referral content
- SEO manipulation

## AI Rules
Content must be:
- Original
- Neutral
- Informational
- Non-promotional

## Mandatory Checklist
- No money-related language
- No promotional intent
- No duplicate content
- Must be educational
- Article sequence number must be assigned before generation
- Every generated article must be added to README with link
